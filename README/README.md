<操作方法>
・キーボード
W,A,S,D		: 移動
LCONTROL	: 忍び足
TAB			: ポーズ

・マウス
左クリック	: 決定
左長押し	: 掘る

・ゲームパッド
左スティック	: 移動
右トリガー		: 忍び足
Aボタン			: 決定、掘る
STARTボタン		: ポーズ
